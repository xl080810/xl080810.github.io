第一节 动物胚胎移植
...

一、概述
...

1.定义
...

　　胚胎移植（embryo transfer，ET）是指将处于某一发育阶段的早期胚胎移植到与其发育阶相对应的同种母畜子宫内，使之受孕并产仔的技术。通常将提供胚胎的母畜称为供体（donor），接受胚胎的母畜称为受体（recipient）。

2.发展概况
...

　　根据其发展历史，大致可以分为三个阶段
初级研究阶段（1890—1950）
技术发展阶段（1950—1975）
应用初始阶段（1975—现在）
近年来，随着体外受精、胚胎分割和性别控制等胚胎生物技术的发展，胚胎移植在畜牧生产中展现出越来越大的作用。

3.意义与作用
...

　　1.充分发挥优良母畜的繁殖潜力，加快品种改良和育种的进程
2.加快某一特定品种扩群速度
3.有利于濒危动物的保护与利用
4.诱发肉畜怀双胎，提高生产效率
5.代替种畜引进
6.保存动物的品种资源
7.防疫需要和克服不孕
8.作为发育生物学和其他胚胎生物技术的研究手段

二、胚胎移植的生物学基础和原则
...

1.生物学基础
...

　　（1）发情后生殖器官的孕向变化
（2）胚胎在发育的早期阶段处于游离状态
（3）移植的胚胎具有免疫耐受性
（4）胚胎的遗传特性不受受体母畜的影响

2.基本原则
...

　　（1）胚胎移植前后所处的环境的同一性
a.供体和受体在分类学上的一致性
b.供体和受体在生理学上的一致性
c.供体和受体在解剖学上的一致性
（2）胚胎移植的时限应在周期性黄体退化之前
（3）胚胎移植的操作过程要进行严格的质量控制

　　总之，按照上述原则，胚胎的移植只是空间位置（现象上）的更换，而不是生理环境（实质上）的改变，这不会影响到胚胎的生长发育（如果有，也是微小的），更不会危害生命。所以，胚胎从一头母畜移植到另一头母畜，或经过运输再移植，能够存活并顺利发育，直到最后由母畜产出，成长为新个体。

三、胚胎移植的操作程序
...

　　主要包括：
胚胎的收集
胚胎的检查
胚胎的培养与保存
胚胎的移植

1.胚胎移植的过程
...

　　由于进行胚胎移植时希望一次从供体母畜得到多个胚胎，所以要对其进行超数排卵处理。同时，胚胎移植成功的先决条件是供体和受体必须同期发情，故需对其进行发情同期化处理。

2.供体和受体的准备
...

　　供体：选择的供体不但应有种用价值，而且生殖机能正常，经超数排卵处理后，可收集得到较多的胚胎。对于牛、羊、猪等家畜，需在两个月以上才宜作为供体。
受体：每头供体需准备数头受体。受体母畜可选用非优良品种的个体或本地家畜，但亦应具有良好的繁殖性能和健康状态，体型中等偏上。当选用本地家畜作为受体时，供受体的体型不宜相差太大，否则会因发生难产而前功尽弃。

3.超数排卵
...

　　在母畜发情周期的适当时间，利用外源促性腺素GTH，增进卵巢的生理活性，激发较多个卵泡在一个发情期中成熟排卵，使其比在自然条件下排出较多的卵子，从而获得更多胚胎的技术，称为超数排卵，简称超排（superovulation）。

　　（1）母牛的超数排卵

　　FSH减量注射法 在母牛发情后第8～13天每天肌注FSH2次，连续4天在FSH开始注射后第3天宫注氯前列烯醇2～3mg，48h后再肌注hCG（2000IU）或LH（500IU），接着间隔12h输精2次。如在发情周期第16天开始处理，则不必宫注氯前列烯醇。如用宁波动物激素厂生产的FSH，注射量为：75IU2，50IU2，25IU2和12.5IU2。如用中国科学院动物研究所提纯的FSH，注射量为：2mg2，1.5mg2，1.0mg2，0.5mg*2。

　　PMSG一次肌注法 在发情周期第8～13天一次肌肉注射PMSG2500～3000IU，48h后 再宫注氯前列烯醇2～3mg。发情后18h肌注等量的PMSG抗血。若在发情周期第16天肌注PMSG，则不必宫注氯前列烯醇。观察到发情后4～6h进行第1次输精，间隔12h后进行第2次输精。

　　（2）母羊的超数排卵

　　孕激素+PMSG或FSH 用甲孕酮（60mg）或氟孕酮（30 ～40mg）制成的阴道海绵栓处理母羊12～14天，山羊14 ～18天，或皮下埋植18-甲基炔诺酮。在处理结束前的1 ～2天，一次肌注PMSG1000～2000IU，或用FSH300IU分3天6次或4天8次皮下注射。

　　PGF​~2a~​+PMSG或FSH 在发情周期第10～13天 （ 绵 羊 ）或 13 ～ 16 天（ 山 羊） ，一次肌注PMSG1000～2000IU，或多次注射FSH300IU，开始处理后的48h，肌注氯前列烯醇0.15～0.25mg。如不明确发情周期，则需间隔10天注射2次氯前列烯醇，GTH在第2次注射前2天开始处理。

　　单独注射FSH或PMSG 在 绵 羊 发 情 周 期 的第13天山羊发情周期的第17天，肌注PMSG或多次注射FSH。山羊需在发情后肌注1000IUhCG或LH 150大鼠单位。

4.同期发情
...

　　通过用激素或其他方法处理，使处于不同阶段的母畜集中于一定时间发情（2-3天内）

　　理论依据 ： 母畜的发情与周期性变化受体内激素调节。主要是促性腺激素和孕激素调节，使卵泡与黄体交替发生。关键是控制黄体。

　　（1）羊的同期发情处理

　　方法1:在任意一天对全体羊注射一次PG，剂量：0.1mg/只，目的是使处在妊娠期的羊流产，第二次注射PG（0.1mg/只） 24小时后便发情，使全体羊的发情同步差只允许在20～24小时内。

　　方法2：在任意一天对羊全部注射PG（0.1mg/只）防止怀孕出现，再在羊阴道内放置孕酮CIDR栓（为0天），第14天上午注射PMSG（2001I.U./只）第16天时取出CIDR栓，在24小时后可发情。

5.胚胎的收集
...

　　指利用冲洗液将胚胎由生殖道中冲出，并收集在器皿中的过程。胚胎收集有手术和非手术两种方法。前者适用于各种家畜，后者仅用于牛、马等大家畜，且只能在胚胎进入子宫角以后进行。

　　收集时间不应早于排卵后的第1天,亦即最早要在发生第一次卵裂之后，否则不宜辨别卵子是否已受精。

　　一般是在配种后3～8天，发育至4～8个细胞以上为宜。牛胚胎最好在发育至晚期桑葚胚或早期囊胚进行收集和移植（配6～8天）。

　　（1）牛的胚胎收集

　　手术法 适用于发情后5d前的母牛冲卵。一般在牛的腹侧肷部做一个10cm左右的切口，然后用手将输卵管引出，在伞部插入一条导管，从宫管连接部注入冲卵液10～15ml。

　　非手术法 采卵前需注入3～5ml 2%普鲁卡因做硬膜外腔麻醉（或肌注1 ml 2%的静松灵）。通常采用二路式导管冲卵器。冲卵器的外管前端连接一气球，当将冲卵器插入子宫内，充气堵住子宫颈内口，以免冲卵液经子宫颈流出。将冲洗液通过内管注入子宫角内，然后导出冲洗液

　　一般每侧子宫需用300～500 ml冲卵液冲洗。冲洗液的导入应顺畅迅速，并尽可能回收全部注入的冲卵液。为此，最好用手在直肠内将子宫角提高并略加按压，以促进冲洗液流出。

　　2）羊胚胎收集

　　麻醉：肌注2%的静松灵做全身麻醉，局部用0.5%普鲁卡因做封闭麻醉。

　　手术部位 ： 在 乳房前的腹中线部，术前需进行手术的常规消毒。

　　冲卵：在发情后2～3d冲卵，从伞部插入导管，从宫管连接部注入冲卵液 5 ～ 1 0 ml （ 上 行 法），也可从反行操作（下行法）。

　　如果在发情后6～7d冲卵，先将导管插入子宫角的基部，再从子宫角的尖端注入冲卵液50～60 ml（下行法），也可反行操作（上行法）。

　　冲卵液常用杜氏磷酸盐缓冲液（PBS）加5%～10%的小牛血清、发情牛血清、胎犊血清或阉公牛血清，也可加0.4%的BSA。

6.胎的检查
...

　　（1）捡胚
回收液少的可直接将其置于平皿中，在体视显微镜下捡胚。回收液多的可通过静置法、漏斗法、过滤杯法等方法预先处理。然后再置于16倍左右的体视显微镜下寻找胚胎。

　　（2）胚胎鉴定
形态学法:
评定内容：卵子是否受精，透明带的形状、厚度、有无破损，卵裂球的致密程度，卵黄周隙内是否有游离的细胞或细胞碎片，细胞大小是否有差异，以及胚胎的结构和发育程度等。根据胚胎的形态特征可将胚胎分为A(优)、B(良)、C(中)、D(差)。

　　荧光活体染色法:
将 二 醋 酸 荧 光 素 （FAD）加入待鉴定的胚胎中，培养3—6分钟，活胚胎显示荧光，死胚胎无荧光。

　　测定代谢活性法:
将待鉴定的胚胎放入含有葡萄糖的培养液中，培养1小时后，测定葡萄糖的消耗量，每培养1小时消耗葡萄糖2—5ug以上者为活胚胎。应注意的是在捡胚及进行胚胎分类时，胚胎均应保持在不低于25。C的温度下。

7.胚胎的冷冻保存
...

　　在室温条件下，胚胎只能存活10～20h。当温度下降至20。C以下时，胚胎即停止发育，存活时间可以延长。牛、羊胚胎在5～10。C可以存活数日。胚胎冷冻保存的方法有三种。

　　冷冻程序为：
胚胎→保存液洗10次→10%甘油保存液→装入细管（15-20分钟）→冷冻仪（-5.0°C）→-5.0°C平衡5分钟→植冰（7-10秒）→继续平衡5分钟→以0.4-0.6°C/分的速率下降→-32°C→平衡5分钟→投入液氮

　　（1）控温冷冻法

　　冷冻液的配制：改进的杜贝磷酸缓冲液+15%FCS或0.4%BSA +1.5mol/L甘油、DMSO或1.81.5mol/L乙二醇。

　　防冻剂的平衡：最初为四步和六步，现多采用三步法和一步法，每步5～10min，平衡后，装入0.25ml的塑料细管中。

　　冷冻：以5。C/min降至-5～-7。C或直接置于-5～-7。C，然后“冰植”。平衡10 min后，以0.1～0.3。C/ min降至-25～-35。C，平衡10 min或立即投入液氮保存。

　　解冻：将胚胎从液氮中取出，在空气中停留10s或直接置于25～35。C的水浴中解冻。

　　除去防冻剂：可按照防冻剂的平衡方法，分步除去防冻剂。亦可直接置于0.3～0.5mol/L的蔗糖中进一步除去防冻剂。目前，通常采用将塑料细管的两端充以0.3～0.5mol/L的蔗糖，中间为含有胚胎的冷冻液，胚胎解冻后直接在塑料细管内除去防冻剂，并马上移植。

　　（2）超速冷冻法

　　冷冻液的配制：改进的杜贝磷酸缓冲液+15%FCS或 0 . 4 % BSA+2.2 ～ 3.5mol/L 甘 油 或 + 0 . 3 ～0.5mol/L的蔗糖。

　　防冻剂的平衡：胚胎先在1.5mol/L甘油或乙二醇中 室温 平衡 1 0 min， 然后 在 4 。 C 冷冻液 中平 衡 5min，而后装入0.25ml的塑料细管中。

　　冷冻：将平衡好的胚胎置于-25～-35。C的低温冰箱中平衡15～30 min然后投入液氮保存。

　　解冻：将胚胎从液氮中取出，直接置于35。C的水浴中解冻。

　　除去防冻剂：将胚胎置于含0.3～0.5mol/L的蔗糖的PBS中平衡5min，然后用PBS清洗2次。

　　（3）玻璃化冷冻保存

　　冷冻液的配制：冷冻的基础液为改进的杜贝磷酸缓冲 液 + 1 5 % FCS 或 0 . 4 % BSA， 防 冻 剂 为 2 5 % 甘 油2 5 % 乙 二 醇 或 丙 二 醇 , 或 2 0 % 乙 二 醇+20%DMSO+0.3～0.5 mol/L的蔗糖。

　　防冻剂的平衡：胚胎先在12.5%甘油+12.5%乙二醇或丙二醇，或10%乙二醇+10%DMSO的平衡冷冻液中室温下平衡2-5 min,然后在4°C的玻璃化冷冻液中平衡1min，或在室温的玻璃化冷冻液平衡30s，而后装入0.25ml的塑料细管中或拉细的开口塑料细管中。

　　冷冻：将平衡好的胚胎直接投入液氮保存。

　　解冻：将胚胎从液氮中取出，直接置于35℃的水浴中解冻 。 如采用开口塑料细管 ， 则 在 含 0 . 3 -0.5mol/L的蔗糖的PBS中解冻。

　　除去防冻剂：将胚胎置于0.3—0.5mol/L的蔗糖的PBS中平衡5 min，然后用PBS清洗2次。

8.胚胎的移植
...

　　胚胎的移植是整个胚胎移植技术的关键一环。胚胎移植的成功率除胚胎质量外，还受操作技术、受体母畜的生殖机能状态和饲养管理等的影响。

　　（1）牛的胚胎移植

　　有手术法和非手术法
（目前均采用非手术法移植）
手术移植 主要用于16细胞以前的早期胚胎。先在牛的腹侧肷部做一10cm左右的切口，然后用手将输卵管引出，从伞部插入一条装有胚胎的硅胶管，再用注射器将胚胎注入到输卵管的壶腹部。

　　非手术移植 主要桑葚胚和囊胚的移植，是牛胚移主要方法。先将胚胎装入0.25 ml的细管内，然后装入胚胎移植枪（由不锈钢内管、内芯和塑料外套组成），通过直肠固定子宫颈将胚胎移入到子宫角的大弯或大弯深处。

　　根据受体牛的发情天数选择胚胎的发育阶段，一般将桑葚胚、早期囊胚和扩展胚分别移植给发情后6d、7d和8d的受体母牛。对于未配种的受体牛可将胚胎移植到黄体侧的子宫角，对已配种的受体可将胚胎移植到黄体对侧的子宫角。

　　（2）羊的胚胎移植

　　通常采用手术法,但近几年随着腹腔镜技术的发展,借助特制的腹腔镜对羊进行胚胎移植已成为一种发展趋势。

　　手术法移植 与采卵程序一样，需进行麻醉、保定、消毒等手术环节，而最终将胚胎移入到输卵管或子宫角的顶端。

　　移胚前的手术过程同供体母羊冲胚手术。将有黄体一侧的子宫角取出，用尖端磨印的16#针头在子宫壁上扎一孔，把装有胚胎的移植器从此孔插入子宫腔内，伸至子宫角尖端后将胚胎轻轻推出即可。受体创口缝合操作步骤同供体手术。

　　非手术法: 用戊巴比妥钠略作麻醉后从阴道插入外径为10mm的导管到子宫颈，然后插入直径为5mm的导管到子宫角。目前多使用腹腔镜，可以获得较好的结果。

第二节 动物体外受精技术
...

一、概 述
...

1.定义
...

　　从狭义上讲，体外受精（in vitro fertilization）是指将动物受精过程中的精卵结合在体外环境下完成的一种现象，包括自然条件下的体外受精（两栖类和鱼类的正常生殖过程）和人为培养条件下的体外受精（哺乳动物）。哺乳动物的体外受精实质将精子和卵子分别从公、母畜体内取出并经过一系列的处理，使精卵在体外条件下结合的过程。

2.体外受精的主要内容
...

　　1）卵母细胞的体外成熟(IVM)
2）卵子精子体外受精(IVF)
3）受精卵的体外培养(IVC)

3.体外受精的意义
...

　　1）提高动物繁殖力的有效措施(精卵)
2）体外受精可为胚胎移植提供充足而廉价的胚胎
3）为其他动物繁殖生物技术，如动物克隆、动物转基因工程和胚胎干细胞等研究提供丰富的试验材料和必要的研究手段
4）有利于揭示受精的本质和机理

4.研究历史
...

　　体外受精是在20世纪80年代初研究成功,并在近10年得到逐步完善的一项配子与胚胎生物技术,1959年，张明觉利用获能处理的家兔精子进行体外受精实验，获得了世界上第一批体外受精的试管小兔;1982年，美国学者Brackett等成功报道世界首例体内成熟的卵母细胞经体外受精获得试管牛；我国体外受精技术的研究起步于80年代后期，但进展很快，已先后在人和牛等8种动物取得成功，已达到国际先进水平。

　　实现良种牛快速大量繁殖首先要做的是 体外受精 和 早期胚胎 的培养。

二、卵母细胞的体外成熟
...

（一）卵母细胞的采集
...

　　1.离体卵巢采卵
母畜屠宰后，取出卵巢放置于消毒的PBS或生理盐水的保温瓶（25~30°C）中，在4~6小时之内送到实验室。从卵巢上回收卵母细胞主要有两种方法：

　　（1）卵泡抽吸法
用5~10ml的注射器连接一支16号针头，将针头插入卵泡而抽吸卵母细胞。此法的优点是容易操作且速度较快，但回收率偏低且易损伤卵球细胞从而影响卵母细胞的质量
（2）卵巢解剖法
首先用手术剪刀将卵巢上的卵泡单个的剪下，然后在盛有回收液的器皿中用消毒刀片划破卵泡，最后将器皿置于体视镜下收集卵母细胞。此法的优点是回收率高，但操作复杂回收速度较慢而且容易造成污染。

　　2.活体采卵
卵母细胞的活体采集是通过体外受精技术生产良种胚胎的一种主要途径。因从屠宰场收集到的卵巢种质一般都相对较差，而通过对良种动物反复进行活体采卵可获得大量种质优良的卵母细胞，其胚胎生产的效率要比超数排卵高出数倍，且对动物的生产性能和生殖机能无太大影响。

　　（1）腹腔镜法
在腹壁切开一小口,插入腹腔镜、操作杆和穿刺针,通过操作杆和穿刺针的配合将卵母细胞抽吸出来。该法的优点是比较直观,容易掌握；缺点是工作量大,对母牛有损伤,不能频繁手术。

　　（2）B超引导法
根据B超屏幕上所显示的卵泡位置进行穿刺而将卵泡抽出。该法的优点是不用手术，操作的速度较快，对母牛的损伤较少，可频繁对母牛进行采卵，亦可对妊娠母牛进行采卵；缺点是操作技术较难掌握，并需要昂贵的超声设备。

（二）卵母细胞的体外成熟
...

　　1．培养液
用于卵母细胞体外成熟的基础培养液有TCM199，Ham’s F-10，NUSC23和MEM等，其中应用最广泛的是TCM199。
2．培养时间
对大多数家畜来讲，卵母细胞的成熟培养时间一般采用22--24小时，但猪多采用40--44小时，马多采用30~36小时。时间过短或过长，都会导致受精率和胚胎发育率下降。

　　3.温度和气相

　　人和啮齿类动物的卵母细胞体外成熟采用37°C，牛、猪、羊和马等均采用38~39°C。
气相一般采用5%的CO2，以维持正常PH值

　　4.培养系统

　　（1）开放培养系统
（2）微滴培养系统
（3）密闭培养系统

（一）定义及意义
...

　　性别控制（Sex control）是指通过认为地干预并按人们的意愿使雌性动物繁殖出所需性别的后代的一种繁殖新技术。性别控制技术在畜牧生产中意义重大。首先，通过控制后代的性别，可充分发挥受性别限制的生产性状(如泌乳)和受性别影响的生产性状(如生长速度、肉质等)的最大经济效益。其次，控制后代的性别比例可增加选种强度，加快育种进程。

第四节 动物克隆技术
...

一、概 述
...

（一）定义
...

　　动物克隆是指动物不经过有性生殖方式而直接获得与亲本具有相同遗传质后代的过程，包括孤雌生殖、卵裂球分离与培养、胚胎分割以及核移植等。将所有非受精方式繁殖所获得的动物均称为克隆动物，将产生克隆动物的方法称为克隆技术。在高等哺乳动物,核移植是生产克隆动物最为有效的技术，根据供体细胞的来源分为胚胎细胞克隆、胎儿细胞克隆和成年动物体细胞克隆三大类。

（二）动物克隆的意义
...

　　1.动物克隆技术能使遗传性状优秀的个体大量增殖，大大加快育种进展；
2.可用来扩大转基因动物的后代数量，提高转基因动物的效率；
3.通过胚胎性别鉴定、再克隆，可生产大量预知性别的动物后代，对于性别控制具有重要意义
4.可用于珍惜和濒危动物的扩繁和保种；
5.满足生物医学研究对实验动物的特殊需要，提高实验的准确性；
6.以此技术为基础，可以研究许多极为重要的基础问题，如动物个体发生的核质互作关系、细胞核的去分化和重新编程问题以及重构胚的线粒体变化、细胞的老化等问题。

（三）动物克隆的历史和现状
...

　　1.德国著名胚胎学家Spemann在1938年对蝾螈受精卵的试验表明蝾螈未分化的卵裂球具有发育的全能性。
2. Briggs和King在1952年成功的完成了蛙细胞核移植试验，从此开辟了高等动物生物学研究的领域。
3. 70年代后，Willadsen等发展和完善了胚胎的切割技术，从此，胚胎分割和卵裂球的分离技术成为了研究热点。
4.80年代后，多数动物的早期胚胎细胞核移植均已取得成功，人们逐步将注意力转向囊胚内细胞团细胞、胚胎干细胞和胎儿细胞的核移植以及继代克隆上面。
5. 1997年，Wilmut等用6岁的绵羊乳腺细胞克隆出了世界上第一例成年动物体细胞的绵羊—“多莉”。哺乳动物体细胞克隆的成功使克隆技术进入了一个划时代的发展阶段。

二、孤雌生殖
...

　　孤雌生殖（parthenogenesis）是指卵子不经受精，直接产生个体的繁殖方式。由于孤雌生殖的个体直接来自卵子，没有经过受精过程导致的遗传重组，其遗传物质与卵子完全一致，故而孤雌生殖也是动物克隆的一种方式。

四、细胞核移植
...

（一）定义
...

　　细胞核移植包括胚胎细胞核移植和体细胞核移植。胚胎细胞核移植又称胚胎克隆，它是通过显微操作将早期胚胎细胞移植到去核卵母细胞中构建核子的生物技术。
体细胞核移植技术又称体细胞克隆，它是把分化程度较高的体细胞移入去核卵母细胞中，构建新合子的生物技术。

（二）核移植的操作方法
...

　　1.供体细胞的种类
（1）胚细胞
（2）体细胞
（3）胚胎干细胞

　　2.受体细胞的去核技术
受体细胞去核方法主要有两种方法
一是：透明带切开法
二是：盲吸法（一步法）

　　3.核卵重组技术
按供体核移入部位的不同氛围卵周隙注射和胞质内注射
（1）卵周隙注射 在显微操作仪下,用去核吸管吸取一枚分离出的完整卵裂球,注入去除核的受体卵母细胞的卵周隙中。
（2）胞质内注射 先将供体细胞的核膜捅破，形成核胞体，再将核胞体直接注入到去核卵母细胞的细胞质中,然后进行激活处理。

　　4.重组胚的融合技术
（1）仙台病毒法
（2）电融合法

　　5.重构胚的培养
（1）体内培养
（2）体外培养

　　2.供体细胞的细胞周期

（1）胚胎细胞核 供体核大多处于S期。
（2）体细胞核 供体核大多处于G0期，效果较好。

第五节 胚胎干细胞技术
...

一、概述
...

(一)胚胎干细胞的生物学特点
...

　　1．来源
胚胎干细胞(embryonic stem cells,ES细胞)主要来源于:
囊胚的内细胞团(inner cell mass,ICM)、
早期胚胎的种细胞(embryonic germ cells)
胎儿原始生殖细胞(primordial germ cells,PGCs)

　　2．细胞学特点
与普通细胞相比，胚胎干细胞的体积较小，细胞核大，核质比高，核仁明显，体外培养时呈扁平而紧密的多细胞克隆状生长。在正常培养体系中，ES细胞能保持非常稳定的核型。
3．功能
ES细胞注入囊胚腔或与早期胚胎的卵裂球聚合后，再移入子宫后能获得嵌合体后代，并且ES细胞能分化为生殖干细胞。因此，ES细胞在体内能分化为任何一种类型的细胞。在体外，胚胎干细胞在诱导因子的作用下也能实现定向分化。

　　4．分子特征
胚胎干细胞的表面存在阶段特异性胚胎抗原，不同种动物，抗原的种类不同。所有的ES细胞内都含有较高活性的碱性磷酸酶、Oct-4蛋白和端粒酶水平。在胚胎细胞，PGCs和ES细胞中两者的指标均高于体细胞。此外，ES细胞的核型非常稳定。

　　5.ES细胞自我更新的分子机理
ES细胞通过三种信号转导途径维持正常功能。
第一.是LIF，其与受体结合后激活Jak酶
第二.是转录调控因子Oct-4，在ES细胞中始终维持一定的水平
第三.是通过抑制酪氨酸磷酸酶SHP-2和细胞外调节激酶ERK，维持ES的正常功能

（二）胚胎干细胞研究的意义
...

　　1．ES细胞是研究哺乳动物个体发生发育规律极有价值的工具。
2．ES细胞是研究细胞分化的理想手段
3. ES细胞是研究基因功能的首选细胞
4．ES细胞是转基因动物的主要方法之一。
5．ES细胞治疗技术将引起一场医学革命
6．ES细胞可用于研究细胞癌变机理和新药物的筛选

二 胚胎干细胞分离培养的基本程序
...

　　根据所用材料的来源，ES细胞的分离方法有两种：ICM和PGCs的分离和培养法。
分离技术的目标是使具有发育多能性的细胞大量繁殖而不发生分化。

（一）胚胎或胎儿材料的选择和处理
...

　　1．囊胚 由于体内受精的囊胚比体外受精胚胎的ICM细胞数多，因此应尽可能选择体内受精的胚胎来分离ES细胞
2．胎儿性腺脊 哺乳动物的PGCs最先出现在胚外中胚层，然后出现在卵黄囊的后区，最后由肠系膜迁移至原始生殖脊

（二）选择抑制细胞分化的培养体系
...

　　这是分离培养ES细胞的关键环节。培养体系不仅能促进胚胎细胞生长，还能抑制细胞分化，维持未分化状态，保持正常的核形。目前用于分离ES细胞的培养体系有三种。
（1）饲养层培养体系
（2）条件培养体系
（3）培养液中添加分化抑制因子体系

（三）胚胎干细胞的分离
...

　　1．以囊胚为材料
分离ES细胞的第一步是获得ICM。目前有两种方法：免疫外科手术法和机械剥离法

　　2．以胎儿性腺脊为材料
胎儿性腺脊经酶消化后可直接培养在分化抑制培养体系中，也可以用Percoll密度梯度离心法富集PGCs，再放入分化抑制体系中培养。一般每7天传代一次，直至出现细胞形态均一的克隆细胞团，挑选后进行扩增培养。以后的处理与囊胚分离ES细胞的方法相同。

（四）胚胎干细胞的保存
...

　　目前保存ES细胞的方法有两种：
一是常规保存，即通过不断更换分化抑制培养液，以传代培养方式维持ES的不分化状态。
二是冷冻保存,即通过超低温冷冻保存是细胞的代谢活动完全停止,达到长期保存的目的。ES细胞的冷冻方法与普通细胞相同,只需在培养液中加入10%的DMSO可在液氮中长期保存。解冻后ES细胞的功能和形态可维持原来的状态。

（五）胚胎干细胞的鉴定
...

　　通过分离培养获得的ICM或PGCs细胞在传到8～12代时,必须通过以下生化和细胞生物学指标的鉴定才能最终确认为ES细胞系
1．形态
2．阶段特异性胚胎抗原（SSEA）和肿瘤排斥抗原（TRA）
3．酸性磷酸酶（Apase）
4．端粒酶检测
5．核型分析
6．发育分化潜力

　　鉴定发育多能性方法有：
（1）体外诱导分化 把待检测细胞培养在无分化抑制物的普通培养液中，如果细胞能自动聚合并分化形成胚状体，可初步确认为具有发育多能性，是ES细胞
（2）畸胎瘤实验 把ES注射到有免疫缺陷成年小鼠或遗传同质动物的皮下或肾脏、睾丸的被膜下，如果能形成畸胎瘤，也可初步判定为具有多能性
（3）嵌合体实验 把培养的ES细胞与早期胚胎的卵裂球聚合，或直接把ES细胞注入囊胚腔，然后让重组胚胎在体内继续发育，通过检测后代的嵌合状态，分析ES细胞的分化潜力
