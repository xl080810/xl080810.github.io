（一）精子受精前准备

1.动物射精与精子运行特性
...

　　

​

2.精子在雌性生殖道内运行的三个生理屏障和贮存库
...

　　子宫颈、子宫、子宫与!输卵管连接处

　　

​

　　子宫颈

　　子宫颈黏膜上的裂隙、沟槽、隐窝和黏液共同构成错综复杂的体系，阻滞精子和死精子进入子宫。进入子宫颈的精子大部分进入子宫颈“隐窝”形成精子贮存库，然后缓慢释放，少部分精子直接进入子宫。

　　子宫

　　进入子宫的部分精子进入子宫内膜腺，构成第二个屏障和贮存库。

　　子宫与输卵管连接处

　　控制进入输卵管的精子数，使精子逐步流进输卵管，构成第三个屏障。而输卵管壶部和峡部连接处的峡部一侧被看作是第三个精子贮存库。
雄性动物交配后，有数亿个精子进入雌性生殖道，然而由于各级屏障结构的筛选，存放在峡部的精子只有1000～10000个，而进入壶腹部的精子则更少，仅10～100个。

3.精子运行动力
...

　　精于活力

　　主要依靠精子尾部的鞭毛运动。

　　激素的作用

　　来自母体的催产素（OXT）作用于雌性生殖道，使平滑肌收缩频率增加，因而促进了精子的运行。
若在交配或人工授精时粗暴地对待母畜使其受惊吓等，引起β-内啡肽(β-EP)的释放．则会抑制精子运行。
精液中富含由精囊腺分泌的前列腺素（PGF） ，PGF被雌性生殖道吸收后，可促进子宫和输卵管的肌肉收缩

　　生物化学作用

　　微碱性能增强精子活力；
卵泡液、输卵管液等能使精子的活力增强；
腔液的某些酶如清蛋白酶、碱性磷酸酶（AKP）、肽酶的减少，以及黏蛋白和氯化钠浓度的增加，也能促进精子运行。

　　免疫学作用

　　雌性生殖道如果含有抗精子或精清的抗体，可与精液或精于抗原发个特异性免疫反应，引起精子发生凝集反应或失去活力。

4.精子获能
...

　　概念

　　射出的精子必须要在雌性生殖道内渡过一段时间，以获得穿入卵子的能力。这个过程称为“获能”。

　　获能现象

　　超激活运动

　　获能因子

　　体内：αβ淀粉酶、雌激素、溶菌体酶、β葡萄糖苷酸酶。
体外：咖啡因、肝素、BSA、钙离子载体

　　获能机制

　　精子表面的去能因子（如附睾蛋白和精浆蛋白）的去除，质膜上的某些受体暴露；
细胞内外的离子交换，精子质膜发生超极化，改变其静止的潜能；
一些白蛋白能够除去质膜中的胆固醇，刺激离子通道，
腺苷酸环化酶激活；
精子内cAMP浓度升高，蛋白激酶活化，促进精子质膜结构和成分的改变，引起膜的稳定性下降、通透性增加和代谢增强，有利于精卵结合和顶体反应的发生。

（二）卵子的受精准备
...

　　是指卵子在输卵管中转运及壶腹部停留期间，进一步完成核、质、膜的成熟，为能够顺利受精创造条件的过程。这些过程包括：
暴露透明带表面糖残基，增强对精子的识别和结合能力；
卵周隙增大和质膜微绒毛的竖起，有利于与精子融合；
皮质颗粒的进一步外移，促使皮质反应的迅速发生；
线粒体的重新分布，能够迅速调整代谢能量的供给；
纺锤体的转向，将有利于第二极体的排出等。

二、受精概念
...

　　两性配子（精子和卵子）相互作用，经融合产生新的个体即合子（zygote）的过程。受精始于精子与卵子相遇，至于两性原核合并形成合子的核。受精过程包括精子穿过放射冠和透明带、精子与卵黄膜融合并随之进行阻止多精入卵反应、原核形成及融合等步骤。

三、受精过程
...

1.精子穿过放射冠
...

　　精子头部质膜上的透明质酸酶和某些糖苷酶的水解作用，能够溶解卵丘细胞和放射冠的细胞间质，在精子的机械运动作用下，共同承担精子穿入卵丘细胞层和放射冠的任务。

2.精-卵识别和结合
...

　　配体与受体
精子头部配体与透明带的受体分子参与精子与卵子的识别和结合，具有种特异性。
主要配体分子
初级配体，位于精子头部质膜，有类凝集素蛋白类（如P59）、糖胺转移酶类（如-N-乙酰基半乳糖胺转移酶）、糖苷酶类（如，-D-半乳酸糖苷酶）等。
次级配体，位于顶体，如顶体酶原/顶体酶等。
主要受体分子
初级受体ZP3。
次级受体ZP2。

3.顶体反应
...

　　当精子与卵子发生初级结合后，ZP3诱导精子发生顶体反应。
顶体反应是包裹精子头部的质膜与顶体外膜融合和破裂，释放顶体内容物的过程。

4..精子穿入透明带-酶作用学说
...

　　精子的次级配体（顶体酶原）与透明带次级受体ZP2发生牢固的结合；
结合后的顶体酶原转化为具有酶活性的α-顶体酶，它能够水解和软化透明带；
α-顶体酶进一步自溶转化为β-顶体酶，仍具有水解特性，但不具结合特性；
透明带可加速β-顶体酶的降解作用；
只要精子顶体内膜上存在顶体酶原，它与透明带结合、转化与降解过程将持续进行，直到进入卵周隙为止。

5.精-卵融合
...

　　精子进入卵周隙后，很快附着于卵子的微绒毛上；
精子头部迅速平卧，以顶体后区的质膜与卵质膜接触；
卵细胞的微绒毛将精子头部包裹并发生质膜间的相互融合；
卵子将整个精子拖入卵细胞质；
精子进入卵细胞质后，只保留DNA，而其它成分被降解。

6.阻止多精子入卵反应
...

　　皮质反应

　　当精子与卵黄膜接触时，在接触点膜电荷发生改变并向周围扩大，整个膜持续去极化，卵黄膜下的皮质颗粒向卵子表面移动，在Ca2+的作用下，皮质颗粒与卵黄膜融合，以胞吐方式将其内容物排入卵周隙。

　　透明带反应

　　皮质颗粒释放类胰蛋白酶以及阻止顶体酶作用的物质，破坏透明带ZP2的结构，改变了透明带的性质。
卵黄膜反应

　　皮质颗粒的膜与卵质膜组合成新的受精卵膜。
皮质颗粒所释放的黏多糖与卵黄膜表面紧密相连，在卵子周围又形成一保护层（透明膜），改变了卵子表面结构。

7.原核形成
...

　　精子解凝集，染色质的周围出现大量的内质网，并逐渐包围精子的染色质，构建新的核膜囊泡，而形成雄原核。

　　排出第二极体的卵子单倍体核物质，以雄原核形成相似的方式形成核膜，包绕卵子的染色质，而形成雌原核。

8.原核融合
...

　　两原核向中央靠拢并紧密接触和融合，染色体重新组合，并准备进行第一次有丝分裂，形成一个称为“合子”的单细胞胚胎。

9.异常受精
...

　　多精子受精：2或3个以上的精子同时接近而进入卵黄。鸟类和爬行类普遍。

　　双雌核受精：额外的原核来自雌性（双雌核），由某一次减数分裂时未排出的极体形成。（猪、田鼠）

　　雌核或雄核发育：精子入卵后，只有一个精核或卵核发育，另一个不发育。

　　孤雌生殖：有性生殖的动物卵子不经过精子的受精作用而被激活，并开始无性生殖过程。无脊椎动物中较常见，脊椎动物中除哺乳类外均有孤雌生殖现象发生。孤雌生殖亦可由实验方法刺激诱导而成。

四、受精过程所需的时间
...

　　精子从进入卵子到第一次卵裂的间隔。
猪：12-14h
绵羊：16-21h
牛：20-24h
兔：12h
人：36h

第二节 妊 娠

一、妊娠的识别与建立
...

　　妊娠是哺乳动物所特有的一种生理现象，是自卵子受精开始一直到胎儿发育成熟后与其附属物共同排出前，母体所发生的复杂生理过程。

1.孕体
...

　　胎儿、胎膜和胎液的复合体，是非常活跃的激素生产单位，在妊娠的起始、维持和终结方面，很大程度上起着主导作用。

2.妊娠的识别
...

　　母体和孕体间联系：

　　 妊娠初期，孕体产生信号 (激素)，传递给母体；
母体产生一定的反应，识别胎儿的存在。

　　妊娠识别信号：

　　 牛 :13-17d，牛滋养层蛋白(bTP)，抗溶黄体作用。
绵羊:14d ,羊滋养层蛋白-1（ OTP-1），抗溶黄体作用；15d 绵羊绒毛膜促性腺激素(OCG)，刺激黄体形成。
猪:11-13d 雌激素（E），抗溶黄体作用。

　　妊娠识别时间：

　　 早于周期黄体消失时间
牛在配种后16－17d

3.妊娠的建立
...

　　在孕体和母体之间产生了信息传递和反应后，双方的联系和相互作用便会通过激素形式或其他生理因素为媒介而固定下来，从而确定妊娠的正常开始。

二、妊娠期
...

　　妊娠(Pregnancy)是指从受精开始，一直到临产的生理变化过程。分胚胎期和胎儿期。

　　

​

三、妊娠维持
...

1.生殖激素的作用
...

　　孕激素（猪、兔）
维持子宫继续发育
降低子宫肌的活动
雌激素
与孕激素协同增加子宫血管的分布
促进子宫内膜的分泌机能
形成子宫PGF贮库

2.黄体机能的维持
...

　　功能性黄体在妊娠60d内都是必须的。
孕体：产生E，P转变为E。
促黄体复合物：黄体组织内LH促类固醇、GnRH促胆固醇合成。
胎盘和肾上腺分泌足量的孕酮

3.母体与胎儿间的免疫反应
...

　　胎盘屏障：滋养层、孕酮
免疫耐受：细胞因子
免疫抑制作用

4.神经调节作用
...

　　子宫肌上有α和β肾上腺素能受体。
交感神经通过两种受体影响子宫肌的活动。
α受体刺激，子宫兴奋，β反之。
妊娠期孕激素使β受体比例加大。

第三节 胚胎与胎盘发育
...

一、胚胎的早期发育
...

卵裂
...

　　是指合子最初的几次细胞分裂，发生在输卵管内，所产生的子细胞称为卵裂球。卵裂的结果，细胞数量增多，但细胞体积逐步减少。
第一次:从动物极通向植物极穿过雌雄原核所在区域分裂。
第二次:同第一次卵裂面垂直，从动物极延伸到植物极。
第三次呈水平方向，在动植物极之间分裂。

桑葚胚和囊胚
...

1.桑椹胚
...

　　经五次卵裂后胚胎变成32细胞，形成像桑椹样的致密细胞团，称桑椹胚

2.囊胚
...

　　桑葚胚继续发育，形成中空的胚胎称为囊胚腔。囊胚由周围的滋养层
细胞和内细胞团组成。囊胚腔一旦形成便持续扩张，引起囊胚体积增
加，最终透明带破裂，胚泡孵出，形成孵化囊胚。

3.早期胚胎的迁移
...

　　胚胎在脱出透明带之前，处于游离状态。胚胎发育所需要的营养，来自子宫乳。
不同的物种，均有发生。单胎动物，胚胎内迁移很少。绵羊高于牛。
不同的动物，选择定位的位置不同。
马：子宫角尾端下方；
反刍动物：子宫角中部，且有特殊的定位面。

二、胚泡附植与胎盘形成
...

　　胚胎的附植：胚胎一旦在子宫中定位，便结束游离状态，同母体建立紧密联系的过程称为着床（imbedding）或附植（implantation）

附植类型
...

　　中央附植（表面附植）：附着前囊胚略有扩张，滋养层大部分暴露在子宫腔内（兔、狗）
偏心附植：胚胎可在子宫上皮内形成一个附着室，胎盘表面与子宫基质相对（小鼠、大鼠）
部分间质附植和间质附植：囊胚穿透子宫上皮，侵入子宫内膜与子宫基质接触（豚鼠、黑猩猩和人)

附植时子宫的变化
...

　　附植前，子宫肌肉活动和紧张度减弱，有助于囊胚在子宫内存留。
子宫上皮血液供应增加，子宫内膜腺体和表上皮分泌活动加强，供给胚胎营养。
子宫液成分变化：如促囊胚蛋白、球蛋白。

胎盘的形成
...

　　胎膜：胎儿的附属膜，是胎儿本体以外包被着胎儿的几层膜的总称。
胎液：羊膜囊的羊水和尿囊内的尿水

1.胎膜
...

　　卵黄囊：由胚胎发育早期的囊胚腔形成。只在早期阶段起营养交换作用。尿膜出现后即被替代。
羊膜：最内层膜，由胚外外胚层和无血管的中胚层形成。羊膜腔。
尿膜：胚体外临时膀胱。血管。
绒毛膜：胚胎最外层膜。绒毛，富含血管网。
脐带：联系胎儿和胎盘的束状组织，被覆羊膜和尿膜，其中有脐动脉、脐静脉、卵黄囊的残迹和脐尿管。

2.胎液
...

　　羊水：最初羊膜细胞分泌，之后大部分来源于母体血液，发育的胎儿也排泄一部分产物。
防止胎儿受压迫；
避免皮肤与羊膜粘连；
使绒毛膜与子宫内膜密切接触，利于附植；
利于子宫颈扩张及产道的润滑。

　　尿水：胎儿的尿液和尿囊上皮分泌物。
扩张尿囊，使绒毛膜与子宫内膜紧密接触；
贮存胎儿的排泄物；
维持胎儿血浆渗透压。

三、胎盘类型及其功能
...

　　胎盘：由尿膜绒毛膜和子宫黏膜发生联系所形成的构造。其中，尿膜绒毛膜部分称为胎儿胎盘，而子宫黏膜部分称为母体胎盘。

（一）胎盘的类型
...

按绒毛分布
...

1.弥散型胎盘
...

　　绒毛分布在整个绒毛膜表面。
子宫黏膜上皮向深部凹入形成腺窝，绒毛插入此腺窝内，又称上皮绒毛膜胎盘。
猪、马、骆驼

2.子叶型胎盘
...

　　绒毛集中，形成绒毛丛，呈盘状或杯状凸起（胎儿子叶）
母体子宫内膜的特殊突出物——子宫阜（母体子叶）
牛子宫阜凸出
羊子宫阜凹陷

3.带状胎盘
...

　　绒毛聚集在绒毛囊中央，形成环带状。
噬血器：母体血液的渗出物组成，与绒毛膜细胞紧密接触，似有吸收红细胞的作用。
猫、犬

4.盘状胎盘
...

　　绒毛膜上的绒毛集中，局限于一圆形区域.
绒毛直接侵入子宫黏膜下方血窦内，又称血绒毛型胎盘。
啮齿类、灵长类

按母体与胎儿真正接触的细胞层次
...

　　上皮绒毛型
结缔组织型
内皮绒毛型
血绒毛型

（二）胎盘的功能
...

　　运输：单纯弥散；加速弥散（细胞膜上有特异性的载体）；主动运输；胞饮作用（免疫活性物质
代谢：有800-1000种酶，含氧化还原酶、转移酶、水解酶、异构酶、溶解酶及综合酶6大类。
内分泌：合成PMSG、PRL、P。
免疫：滋养层组织抗原性弱;唾液粘蛋白;孕酮;母体内分泌（hCG）

（三）胚胎的营养来源
...

　　桑椹期：自身的卵黄质来维持
囊胚期：主要来源于“子宫乳”
胚胎附植期：
吸收和吞食子宫乳；
滋养层摄取子宫上皮的细胞碎屑；
正在形成的胎盘传递营养。
完成附植和胎盘形成后：胎盘

第四节 妊娠母畜主要生理变化
...

一、体重和膘情变化
...

　　食欲增进，体重增加，被毛光润（妊娠诊断的标志之一）。
饲养水平不高导致妊娠后期体重减轻明显，甚至造成胚胎死亡。
适配年龄晚于性成熟年龄
缺Ca、P
循环、呼吸、消化、排泄系统（妊娠期酮血症、血凝增强）

二、生殖器官的变化
...

　　卵巢:妊娠黄体，发情周期中断。
子宫：容积增大。
增生：附植前（孕酮），血管分布增加，腺体增长，卷曲。
生长：附植后和怀孕前期（E2和P），肌纤维肥大，变长。
扩张：怀孕后期，生长减慢，壁扩展变薄。
子宫颈：子宫颈栓。子宫颈的括约肌收缩很紧，使子宫颈管处于完全封闭的状态（防止外物进入子宫内，起到保护胎儿免受外界刺激的作用）。
阴道和阴门：妊娠初期，阴门收缩紧闭，阴道干涩；妊娠末期，阴唇、阴道水肿，柔软有利于胎儿产出。妊娠后阴道黏膜的颜色变苍白，黏膜上覆盖有从子宫颈分泌出来的浓稠黏液。

三、生殖激素的变化
...

　　GTH：FSH变化不明显；LH逐渐升高后逐渐降低。
性腺激素：雌激素黄体和胎盘也可分泌，而且孕激素可变为雌激素。孕激素黄体、胎盘组织和肾上腺分泌，临近分娩前孕激素才急剧下降或消失。
分娩前激素变化：三高（雌激素、催产素、前列腺素）一低（孕酮）。

第六节 胚胎发育和妊娠诊断
...

一、 早期诊断的意义
...

　　妊娠诊断：指通过手工或仪器设备检查母畜是否怀孕，怀孕的阶段以及怀孕是否正常的方法。

　　目的意义

　　 怀孕：加强管理
空怀：及时配种
方法必须：早期、准确、简单，对妊娠无影响

二、妊娠诊断的方法
...

（一）外部观察法
...

　　通过观察母畜的外部征状进行妊娠诊断的方法。此法适
用各种母畜。其缺点是，不易做出早期妊娠诊断，对少数生
理异常的母畜易出现误诊，因此常作为妊娠诊断的辅助方法。

　　母畜妊娠后，发情周期停止，食欲增加，膘情好转，被毛光亮；行动谨慎，怕拥挤，易离群。
妊娠初期，外阴部干燥收缩、紧闭、有皱纹，至后期变为水肿状态。
妊娠后一定时期(牛、马5个月左右、羊3－4个月、猪2个月左右)，可见腹围增大，且向—侧伸展(牛、羊向右腹部突出，猪腹部下垂，马通常左腹部突出)。
在饱食或饮水后，可见胎动，能听到胎儿心音。
妊娠后期，乳房发育膨大，有时四肢下部及下腹出现浮肿，排粪尿次数增多。
在产前l周左右，能从乳房中挤出清亮的乳汁。

（二）腹部触诊法
...

　　是用手触摸母畜的腹部，感觉腹内有无胎硬块或胎动来进行妊娠诊断的方法。此法多用于猪和羊。腹部触诊法只适于妊娠中后期。
A、猪的触诊 先用抓痒法使猪侧卧，然后用—只手在最后两对乳头的上腹壁下压，并前后滑动，如能触摸到若干个大小相似的硬块(胎儿)，即确诊为妊娠。
B、羊的触诊 检查者面向羊的尾部，用双腿夹持母羊颈部进行保定，然后将两手从左右两侧兜住羊的下腹部并前后滑动触摸，如能摸到胎儿硬块或黄豆粒大小的胎盘子叶，即为妊娠。

（三） 阴道检查法
...

　　用开膣器打开母畜的阴道，根据阴道黏膜和子宫颈口的状况进行妊娠诊断的方法。
母畜妊娠后，阴道黏膜苍白，表面干燥、无光泽、干涩，插入开膣器时阻力较大。子宫颈口关闭，有子宫栓存在。随着胎儿的发育，子宫重量的增加，子宫颈往往向一侧偏斜。
此法的不足点是当母畜患有持久黄体、子宫颈及阴道有炎症时，易造成误诊。阴道检查往往不能做出早期妊娠诊断，如果操作不慎还会导致孕畜流产，故应做为一种辅助妊娠诊断法。

（四） 直肠检查法
...

　　是用手隔着直肠壁触摸卵巢、子宫、子宫动脉的状况及子宫内有无胎儿存在等来进行妊娠诊断的方法，适合于大家畜。其优点是诊断的准确率高，在整个妊娠期均可应用。但在触诊胎泡或胎儿时，动作要轻缓，以免造成流产。

（五）实验室诊断法
...

1.血、乳中孕酮水平测定
...

　　妊娠期Ｐ水平比发情期和黄体期都高，是诊断妊娠的很好依据。
一般在母畜配种后一个情期左右的时间，进行血液或乳中P4水平检测。如果P4水平高，则表示可能怀孕。其他时候也可以进行检测。
奶牛乳中孕酮水平：
P>7 ng/ml，妊娠；
5<P<7 ng/ml，为可疑；
P<5 ng/ml，为未孕。

2.免疫学诊断法
...

　　根据免疫学抗体与抗原能特异结合的原理，找到妊娠雌性产生的特殊物质作为抗原或抗体，再制备相应的抗体或抗原与之发生反应，从而诊断妊娠。
血液中与妊娠有关的激素：
人：早孕试纸，检测HCG；
反刍动物：可以检验Trophblastin, 或干扰素τ。

3. 超声波检测
...

　　通过直肠放置探头，于屏幕观察胎儿的存在与否。
牛：在妊娠25d以后。
马：在妊娠14d以后。
羊：妊娠30d后，40-50d后可分出胎儿数量。
猪：妊娠32d后进行，就能估测出胎儿数目。
