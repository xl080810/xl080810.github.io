第一章 动物生殖器官构造与功能
...

第一节 动物生殖器官概论
...

一、生殖器官的发生与发育
...

1.生殖器官的起源
...

　　受精卵→桑椹胚→囊胚→孵化囊胚→内/中/外胚层 　　中胚层→前肾→中肾→后肾→肾 　　中肾→尿生殖嵴→中肾嵴和生殖嵴→生殖腺

2.尿生殖嵴的发育
...

　　中肾和中肾管在发育过程中其腹侧出现纵行隆起嵴，称为尿生殖嵴。其中，外侧叫中肾嵴，内侧叫生殖嵴。 　　随着生殖嵴的细胞层数增多，其表层形成生殖上皮，其内部形成了生殖腺的原始胚基。 　　当胚胎向雄性分化时，生殖腺发育为睾丸，当胚胎向雌性方面分化时，生殖腺则发育成卵巢。

3.雄性动物生殖器官
...

　　睾丸----附睾---输精管----副性腺（精囊腺，前列腺，尿道球腺）---阴茎及包皮

4.雌性动物生殖器官
...

　　卵巢---输卵管---子宫（子宫角，子宫体，子宫颈）--阴道---外生殖器

二、生殖器官的分化
...

1.雄性生殖器官分化
...

　　① 生殖腺发育形成睾丸，生殖腺内上皮细胞排列形成精细管索→睾丸网
②中肾管→附睾管和输精管→精囊腺、前列腺和尿道球腺
③ 生殖结→阴茎

2.雌性生殖器管的分化
...

　　① 生殖腺发育形成卵巢，生殖腺表面部分形成皮质索，中央部分形成髓质索
② 皮质索→原始卵泡
③ 缪勒斯管→子宫、输卵管、阴道。

第二节 雄性的生殖器官及机能
...

　　组成：睾丸、附睾、输精管、副性腺、尿生殖道、阴茎、阴囊。
各种动物其生殖器官的形态及功能相似，但其大小、重量又有其各自特点。

一、 睾丸
...

1.形态和位置
...

　　睾丸是雄性动物的生殖腺。各种动物的形态较相似（卵圆形或长卵圆形），分为游离缘和附着缘（附睾附着侧），成对存在，分别位于阴囊的两个腔内。

　　牛、羊睾丸的长轴与地面垂直，头向上，尾向下； 猪睾丸长轴呈前低后高倾斜

2.睾丸的大小及重量
...

　　不同动物、同种动物不同生理阶段的动物睾丸大小、重量不同

　　季节性繁殖动物在配种季节睾丸重量增加
非季节性繁殖动物随年龄的增加睾丸增大

3.隐睾
...

　　睾丸在胎儿期存在于腹腔内，随着出生，正常情况下应沿腹股沟进入阴囊内。隐睾（cryptorchids): 动物出生或成年后睾丸未能进入阴囊、仍留在腹腔内，该动物不做种用。

4.睾丸的组织结构
...

　　睾丸表面被以浆膜，即固有鞘膜，其内为致密结缔组织构成的睾丸白膜，睾丸白膜从睾丸头向睾丸实质部伸入结缔组织索，构成睾丸纵隔，向四周伸出的放射状结缔组织小梁，为中隔，将睾丸分成许多睾丸小叶，每个小叶内由2－3条精细管组成，精细管在小叶顶端汇合成直细管，形成睾丸网，盘曲成附睾头。

　　

​

　　精细管是精子产生的场所 ， 其内充满液体。各种动物精细管长度和重量各不相同

5.睾丸的功能
...

　　合成并分泌雄激素：精细管之间的间质细胞分泌雄激素； 支持细胞分泌抑制素、激活素等；

　　精子生成： 由精细管生殖上皮的生殖细胞生成，精子的日产量随动物品种不同而不同

6.睾丸支持细胞的主要作用
...

　　参与形成血睾屏障 ，创造了相对稳定的生精环境
支持生精细胞，促进生精细胞转位及精子释放
分泌蛋白质及细胞因子
营养生精细胞

二、 附睾
...

1.形态和结构
...

　　附着于睾丸的附着缘 ，分为头、体和尾 ；

　　附睾头由 13- - 20 条睾丸输出管组成。

　　附睾管的长度因动物不同而不同，牛为 30- - 50m ；马为 80m ；猪和羊为 50- - 60m

2.机能
...

　　促进精子成熟： 附睾产生依赖于雄激素的蛋白，使精子获得结合透明带的能力；
吸收作用： 附睾头和附睾体的上皮细胞能吸收精液中的水分和电解质，提高精子浓度；

　　运输作用： 通过附睾管平滑肌收缩和纤毛的摆动，将附睾头的精子送至附睾尾；
贮存作用： 附睾管的微酸环境（ pH=6.2- - 6.8) 、高渗透压（ 400 ）、低温等使精子代谢和活动维持在较低水平，因而可在附睾贮存较长时间。

三、阴囊
...

　　是皮肤、肉膜、睾外提肌、筋膜和总鞘膜的总称。有两个腔 ，两个睾丸分别位于其中。
具有温度调节能力，保护精子正常生成。
温度高时，肌肉和皮囊松弛，睾丸下降悬于阴囊中；
温度低时，肉膜和睾丸外提肌的收缩作用使睾丸上提

四、输精管
...

　　由附睾管延续而来，输精管有一膨大部，而且有分泌腺，参与形成精液 。
输精管的平滑肌非常发达，管径小， 精液输出时具有强有力的收缩作用。

五、副性腺
...

1.副性腺的组成
...

　　由精囊腺、前列腺和尿道球腺组成。

　　 　　精囊腺 ：精囊腺成对存在于输精管末端的外侧，呈蝶形覆盖于尿生殖道骨盆部。猪的精囊腺最大 ；精囊腺和输精管共同开口于尿生殖道骨盆部精阜。

　　前列腺： 位于精囊腺的后方，由体部和扩散部组成， 牛和猪体部小，扩散部大；羊无体部； 马前列腺较发达。

　　
尿道球腺：成对存在于尿生殖道骨盆部后端，猪的最大；马的较发达；牛、羊较小；犬无尿道球腺。尿道球腺两侧都有排出管。

2.副性腺的机能
...

　　冲洗尿生殖道： 尿道球腺的分泌物先排出，冲洗残留的尿液；
稀释、营养、活化和保护精子： 　　副性腺的弱碱性环境增强精子活力； 　　精液中的各种营养物质是精子代谢的主要能源物质； 　　副性腺中的大量液体扩大精液量； 　　运送精子： 副性腺分泌液与输精管、附睾等运送精子；
形成阴道栓，防止精液倒流： 尿道球腺分泌物的凝固作用，形成阴道栓，防止精液倒流。

六、尿生殖道
...

　　尿生殖道是精液和尿液的共同通道。由骨盆部和阴茎部组成。其中的精阜是输精管、精囊腺、前列腺的共同开口。

第三节 雌性动物的生殖器官及机能
...

　　雌性动物生殖器官由卵巢、输卵管、子宫、阴道、外阴部组成。上为直肠，下为膀胱。

一、卵巢
...

1.形态和位置
...

　　各种动物的卵巢 成对存在 ，其位置、大小和形状各不相同，与母畜所处的生理阶段有关。

2.组织结构
...

　　卵巢由皮质部和髓质部组成

　　皮质部包在髓质部的外面，内含同发育阶段的卵泡和红体、白体和黄体；

　　髓质部含有许多血管和神经；
卵巢门上有成群较大的上皮细胞（门细胞）分泌激素。

3.机能
...

　　卵泡发育和排卵： 各级卵泡逐渐发育，最终排卵，形成黄体；
分泌雌激素、孕激素和其它激素： 卵泡内膜和外膜细胞合成雄激素，再在酶的作用下转化为雌激素；排卵后的黄体分泌孕激素维持妊娠。此外，还能分泌抑制素、卵泡抑素、卵泡激动素等。

二、输卵管
...

1.位置和形态
...

　　输卵管是卵子进入子宫的通道。输卵管由伞部或漏斗部、输卵管膨大部和输卵管峡部组成。伞部为 接纳卵子的部位；壶腹部是受精的部位；峡部连接输卵管和子宫的部位。

2.组织结构
...

　　输卵管管壁由外向内由外膜、肌层和粘膜组成。输卵管上皮细胞由有柱状纤毛细胞和无纤毛细胞组成，纤毛细胞在伞部特别多，使卵子向子宫方向移动；无纤毛细胞为分泌细胞。

3.机能
...

　　运送卵子和精子。
精子获能、卵子受精和早期受精卵发育的场所。
分泌机能：分泌粘多糖和粘蛋白，为精子、卵子和早期受精卵提供营养。

三、子宫
...

1.位置和形态
...

　　组织结构

　　子宫大部分位于腹腔，少部分位于骨盆腔；背侧为直肠，腹侧为膀胱；

　　前接输卵管，后接阴道。由子宫角、子宫体和子宫颈组成。

　　各种动物子宫形状不同，如：
牛/羊子宫角基部将子宫角分开，“双间子宫”
猪子宫角很长，“双角子宫”
马子宫体较长，子宫角基部无纵隔，“双角子宫”

2.组织结构
...

　　由外向内由浆膜、肌层和粘膜三层组成

3.机能
...

　　贮存、筛选和运送精液
精子获能的场所
胎儿发育的场所
调节卵巢功能，导致发情
